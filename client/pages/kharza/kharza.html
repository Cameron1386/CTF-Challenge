Forensics: Files, Logs, Steganography & Magic Bytes

  • Magic Bytes (File Signatures) – Every file type starts with hidden “magic bytes” that identify what it really is. In CTFs, files may be renamed, but checking these bytes reveals the true format.
  • Hex Editor (Inspecting Raw Files) – Lets you view the raw binary data inside files. Useful for spotting corrupted files, hidden content, or mismatched file types.
  • Log File Analysis – Logs record system and web activity like logins, errors, and requests. In forensics challenges, logs often reveal hidden actions or suspicious behavior.
  • Steganography (Hidden Data in Files) – The technique of hiding data inside images, audio, or other files without changing how they appear. Common in CTFs to hide flags inside normal-looking files.
